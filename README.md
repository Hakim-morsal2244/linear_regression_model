Mission:
This project builds a machine learning model to predict student exam scores based on study habits and lifestyle factors. 
The goal is to understand how variables such as study time, sleep hours, and attendance influence academic performance.
The dataset was obtained from Kaggle and contains multiple features affecting student performance.
The project trains Linear Regression, Decision Tree, and Random Forest models to determine the best predictive model.

Dataset Source:
Kaggle – Student Performance Factors Dataset
